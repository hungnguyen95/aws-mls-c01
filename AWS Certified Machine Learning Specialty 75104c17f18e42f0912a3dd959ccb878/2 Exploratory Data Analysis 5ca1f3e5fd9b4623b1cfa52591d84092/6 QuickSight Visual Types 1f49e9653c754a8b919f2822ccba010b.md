# 6. QuickSight Visual Types

- AutoGraph
- Bar Charts
    - For comparison and distribution (histograms)
- Line graphs
    - For changes over time
- Scatter plots, heat maps
    - For correlation
- Pie graphs, tree maps
    - For aggregation
- Pivot tables
    - For tabular data
- Stories
